# android-map-search

## 📄 프로그램 설명

2주차 어플리케이션에서 local DB를 사용하는 것을, 카카오 로컬 API를 사용하여 서버에서 데이터를 받아오는 것으로 변경한 어플리케이션입니다.

이후 지도 화면을 구현하여, 위치 검색과 지도를 볼 수 있는 어플리케이션을 제작합니다.

    - [X] Retrofit을 사용하여 API를 호출하는 인터페이스 생성하기
