### 1. 카카오 로컬 API 검색
1. **카카오로컬 API 연동**
    - 카카오로컬 API를 사용하여 검색 기능 구현
    - Retrofit을 이용하여 네트워크 통신 설정
    - 검색어 입력 시 API 호출하여 결과를 받아오기

2. **검색 결과 표시**
    - 검색 결과를 RecyclerView를 사용하여 세로 스크롤 형태로 표시
    - 검색 결과는 최대 15개 이상 표시
